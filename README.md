# 🛒 Customer Market Basket Analyzer

A Streamlit-based Retail Analytics and Market Basket Analysis application that helps businesses discover customer purchasing patterns, analyze sales performance, build predictive models, and generate actionable recommendations.

---

## 🚀 Features

### 📊 Dashboard
- Revenue Analysis
- Sales KPIs
- Customer Insights
- Product Performance
- Business Overview

### 📈 Analytics
- Monthly Sales Trends
- Product Analysis
- Customer Analysis
- Revenue Insights
- Interactive Visualizations

### 🔗 Market Basket Analysis
- Apriori Algorithm
- Frequent Itemsets
- Association Rules
- Support, Confidence, Lift
- Product Affinity Analysis

### 🎯 Customer Segmentation
- Customer Clustering
- Spending Analysis
- Customer Value Identification
- Segment Visualization

### 🤖 Machine Learning
- Linear Regression
- Random Forest
- Gradient Boosting
- Model Evaluation
- Sales Prediction

### 💡 Recommendations
- Product Recommendations
- Cross-Selling Opportunities
- Customer Retention Strategies
- Business Growth Suggestions
